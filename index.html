传统抄表的痛点

电力抄表工作面临诸多挑战，我们的智能解决方案帮您应对这些问题

手工录入效率低

抄表人员需现场手写记录，再返回办公室手动录入系统，耗时且容易出错，重复劳动浪费大量时间。

数据录入易出错

人工抄录数字容易看错、写错，尤其在光线不好或表盘模糊时，数据准确性难以保证。

历史数据难追溯

纸质记录分散存放，查询历史数据困难，缺乏统计分析能力，难以发现用电异常。

核心功能

全方位的电表信息管理能力，助力电力抄表工作数字化转型

移动端现场采集

专属移动 APP 支持现场拍照采集，自动压缩上传，支持离线模式，网络恢复后自动同步数据。

AI-OCR 智能识别

采用 Qwen2.5-VL 和 GLM-4V 视觉语言模型，自动识别电表读数，支持批量处理，大幅减少人工录入。

数据统计分析

支持按供电所、日期范围、电表类型多维度统计，自动计算提交量、完成率，快速掌握抄表进度。

多类型电表管理

支持公用表（公变）、专用表（专变）、用户表三种类型，独立表单、独立统计，满足不同业务需求。

权限角色管理

支持管理员和普通用户角色，普通用户只能查看自己的数据，管理员可查看和管理所有数据。

台账数据导入

支持从 Excel 批量导入变压器和电表台账数据，自动建立变压器-电表映射关系，简化数据初始化。

常见问题

关于电表信息管理系统的常见问题解答

系统支持哪些类型的电表？

系统支持三种电表类型：

  • 公用表（公变）：供电所辖区内的公用变压器电表
  • 专用表（专变）：专用变压器配套电表
  • 用户表：终端用户的用电表计

每种类型有独立的表单和统计功能，可根据业务需要分开管理。

OCR 识别准确率如何？

系统采用先进的视觉语言模型进行 OCR 识别：

  • 高精度模型：Qwen2.5-VL-7B-Instruct，识别准确率更高
  • 免费模型：GLM-4.1V-9B-Thinking，性价比高

在光线良好、表盘清晰的情况下，识别准确率可达 95% 以上。系统保留原图，可随时人工复核。

移动端 APP 支持哪些平台？

移动端 APP 基于 React Native + Expo 开发，支持：

  • iOS：iPhone 和 iPad
  • Android：安卓手机和平板

支持离线模式，在无网络环境下可正常采集数据，待网络恢复后自动同步到服务器。

如何部署系统？

系统支持多种部署方式：

  • Docker 部署：提供 docker-compose 一键启动，包含应用和 MongoDB
  • 本地部署：Java 17 + Maven，连接本地或远程 MongoDB
  • 云服务部署：可部署到阿里云、腾讯云等云平台

部署文档详尽，技术支持到位，确保快速上线。

数据安全如何保障？

系统采用多层次安全措施：

  • 身份认证：基于 Session 的用户认证，登录审计记录
  • 权限控制：角色权限管理，普通用户只能访问自己的数据
  • 数据存储：可部署在内网环境，数据不出企业网络
  • 文件安全：上传文件大小限制，自动压缩处理
